>[!info] 定義 簡單句只包含**一個變位動詞**，在文法上它獨立構成一個「子句 (proposition)」。

簡單句可以根據動詞後方接續的成分，分為四種基本擴充句型：

1. **Sujet + verbe (主詞 + 動詞)**：
	- _Les cloches sonnent._ (鐘聲響起。)
	- Il réfléchit.
2. **Sujet + verbe + attribut (主詞 + 動詞 + 屬語/表語)**：透過連綴動詞 (_être, paraître, sembler, devenir_) 連結。
	- _Ces fleurs sont magnifiques._ (這些花很美。)(magnifiques = attribut)
	- Stéphane deviendra avocat. (avocat = attribut)
3. **Sujet + verbe + compléments d'objet (主詞 + 動詞 + 受詞)**：包含直接受詞 (COD) 或間接受詞 (COI)。
	- Le Soleil éclaire **la Terre**. (complément d'objet direct)
	- J'ai écrit **à ma meilleure amie**. (complément d'objet indirect)
	- _J'ai écrit une lettre (COD) à ma meilleure amie (COI)._

4. **Sujet + verbe (+ COD/COI) + compléments circonstanciels (加入情境補語)**：補充時間、地點、原因等資訊。Pour apporter une information sur le lieu, le temps, la cause, etc. Leur place dans la phrase est variable.
	- complément circonstanciel de lieu : 補充地點
		- Beaucoup de gens se promènent **dans les jardins du château de Versailles**.
	- complément circonstanciel de temps : 補充時間
		- **Dimanche prochain**, nous irons visiter le château de Versailles.
	- complément circonstanciel de cause : 補充原因
		- Nous ne sommes pas allés nous promener dans les jardins du château **à cause du mauvais temps**.
	- complément circonstanciel d'opposition : 補充相反情境
		- Nous nous sommes promenés dans les jardins du château de Versailles **malgré le mauvais temps**. 雖然天氣不好，我們還是有去凡爾賽宮散步。
	- complément circonstanciel de but : 補充目標
		- **Pour aller à Versailles**, nous prendrons le train.

>[!warning] Remarques : 簡單句的特殊省略形態 (書中備註) 雖然標準簡單句需要「主詞+動詞」，但有幾種特殊情況會省略主詞或甚至沒有動詞：
>1. **命令式 (L'impératif) 與不定式 (L'infinitif)**：主詞不會被表達出來。
>- _Viens !_ (過來！)
>- _Ne pas entrer !_ (禁止進入！)
>2. **名詞句 / 單詞句 (La phrase nominale / Mot-phrase)**：句子可以僅限於一個名詞或一個名詞群，完全沒有動詞（這在招牌、警告標語或口語中很常見）。
>	- _Attention !_ (注意！)
>	- _Ouverture automatique des portes._ (車門自動開啟。)
